Welcome to the RASH-STORE

RASH-STORE is a back-end API for a store that includes several useful endpoints to perform various operations like listing products, adding products to a cart, and creating orders...

This project was built using the Django framework, so it contains a fully customizable dashboard for admins , making it simple for them to manage the store. Also, I have included a useful browsable API that will help you discover the endpoints without needing Postman or curl commands.

Note: Some endpoints require you to be authenticated to use them.
